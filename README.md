# 🎨 壮脉智承 — 沉浸式壮锦文化课堂

**“壮脉智承”** 是一个融合 **AI + 文化教育 + 创意共创** 的智能体工作流项目，以 **壮族织锦文化** 为核心，通过 **语音、图像、数据库交互与 AI 生成艺术**，为用户带来沉浸式文化体验。

------

## 📖 核心亮点

- **沉浸式学习**：通过图文、音频和互动问答，让用户深入了解壮族织锦文化
- **AI 驱动**：文生图、意图识别、用户创作辅助
- **创意共创**：用户可生成数字壮锦作品并入库、分享
- **社群互动**：线上画廊展示用户创作成果

------

## 🏗️ 功能介绍

- **智能欢迎与引导**
  - 播放欢迎音频，展示欢迎图文
  - 解析用户意图，快速进入探索流程
- **文化探源**
  - 展示壮族织锦起源、历史长河与文化背景
  - 提供互动问答，增强学习体验
- **纹样解密**
  - 查询各类纹样及其象征意义
  - 用户可选择纹样，获取详细解析
- **匠心工艺展示**
  - 展示织造工艺动画与制作过程
  - 结合用户互动，体验手作乐趣
- **创意生成**
  - 用户输入创作主题与创新程度
  - AI 自动生成作品图片，并解读寓意
- **成果展示与分享**
  - 生成文化体验证书
  - 可生成二维码分享作品
  - 用户创作入库并展示在数字画廊
- **社群画廊浏览**
  - 随机展示用户创作的数字壮锦作品
  - 提供作品寓意解读

------

## 🧭 项目架构

- 数据库：存储知识库与用户作品
- 图床：存储生成的数字艺术作品
- AI 模型：文本生成、图像生成、语音合成
- 前端展示：沉浸式文化互动界面
